# butterfly-wings
Frontend for butterfly!!

## Taro Introduce
https://docs.taro.zone/docs/GETTING-STARTED

## Run as H5
### yarn
$ yarn dev:h5
$ yarn build:h5

### npm script
$ npm run dev:h5
$ npm run build:h5

### taro
$ taro build --type h5 --watch
$ taro build --type h5

### npx
$ npx taro build --type h5 --watch
$ npx taro build --type h5
